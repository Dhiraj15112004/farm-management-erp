# Farm Management ERP

A role-based Farm Management ERP developed using Django.

## Features
- Role-based login system
- Admin dashboard
- Master setup
- Crop & season management
- Worker & labor management
- Machinery logs
- Inventory & store
- Finance & accounts
- Sales & harvest
- User management

## Roles
- Admin
- Manager
- Supervisor
- Storekeeper
- Accountant
- Sales
- Field Staff

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run
1. Install dependencies
2. Run migrations
3. Start server using:
   python manage.py runserver
